# Travel ERP

Sistem ERP untuk travel agency. Mendukung Umroh, Haji, Tour, dan bisnis travel lainnya.

## Fitur

- Manajemen Customer (CRM)
- Manajemen Booking
- Paket dan Jadwal
- Operasional (Manifest, Rooming, Flight)
- Keuangan (Payment, Invoice, Komisi)
- HRIS
- Inventory
- Marketing
- Support Ticket
- Multi-Tenant SaaS
- AI Assistant
- Multiple Views (Table, Calendar, Kanban)

## Tech Stack

- Next.js 15
- TypeScript
- PostgreSQL + Prisma
- Tailwind CSS

## Quick Start

```bash
npm install
cp .env.example .env
npx prisma db push
npx tsx prisma/seed.ts
npm run dev
```
