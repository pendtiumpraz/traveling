## Cara Login

1. Buka aplikasi di browser
2. Masuk ke halaman `/login`
3. Masukkan email dan password sesuai tabel di atas
4. Klik tombol "Masuk"

## Reset Password

Jika lupa password, hubungi Super Admin untuk reset password melalui menu User Management.
